# Moqui Naming Conventions
- **Entity Definitions:** Never use the `.ent.xml` suffix. All entity definition files MUST end with `Entities.xml`.
- **Naming Pattern:** The filename should be `[EntityGroup]Entities.xml` (e.g., `HuddleArtifactEntities.xml`).
- **Location:** These files must always reside in the `entity/` directory of a component.
